## Installation

1. Make sure you have installed:
- Python 3.8+
- pip

2. Install dependencies:
```bash
pip install flask flask-socketio psutil
```

3. Start the server:
```bash
python app.py
```

4. Open in a browser:
```
http://localhost:5000
```
